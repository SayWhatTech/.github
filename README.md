# SayWhatTech `.github`

This repository holds **organization-level configuration** for [SayWhatTech](https://github.com/SayWhatTech) on GitHub. Content here applies across the org (not to a single application repo).

## What lives here

| Path | Purpose |
|------|---------|
| [`profile/README.md`](profile/README.md) | **Organization profile** — the README shown on the [SayWhatTech organization page](https://github.com/SayWhatTech). GitHub renders this from a repo named `.github` owned by the organization. |

## Adding more later

You can extend this repo with other GitHub features as needed, for example:

- **Workflows** — reusable Actions under `.github/workflows/`
- **Community health** — `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md` at the org root (same repo)
- **Issue / PR templates** — `.github/ISSUE_TEMPLATE/`, `pull_request_template.md`

See [GitHub’s documentation on organization profiles](https://docs.github.com/en/organizations/customizing-your-organizations-profile) and [creating a default community health file](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## About SayWhat

[SayWhat](https://saywhat.tech) helps communities understand complex documents through clear, searchable answers—starting with HOA governing documents and expanding from there.
