## Community Health

SayWhat Technology welcomes contributions and feedback from the community. As we grow, this repo will house org-wide defaults for:

- `CODE_OF_CONDUCT.md` — expected behavior across all SayWhatTech projects
- `CONTRIBUTING.md` — how to contribute to our repos
- `SECURITY.md` — responsible disclosure policy
- Issue and PR templates under `.github/ISSUE_TEMPLATE/` and `pull_request_template.md`

See GitHub's docs on [default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for how these apply across the org.
